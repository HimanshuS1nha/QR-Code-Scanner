# QR Code Scanner

Scan QR codes easily with this QR Code Scanner app.

## Features

- Scan any QR codes easily.
- Zoom in and Zoom out.

## TECH STACK USED

- React Native Expo
- Tailwind CSS
- Expo Camera
- Expo Router
